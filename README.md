# Wordle Frontend

This project is a React-based Wordle clone. Follow the steps below to install dependencies, run the development server, and build for production.

## Prerequisites
- Node.js (v18+ recommended)
- npm (comes with Node.js)

## Installation
1. Install dependencies:
   ```bash
   npm install
   ```

## Running the App 
1. Start the development server:
   ```bash
   npm start
   ```
2. Set the environment variable REACT_APP_API_URL to point to the back-end.
3. Open http://localhost:3000/ in your browser. The app reloads automatically when you save changes.

## Testing
Run the interactive test suite:
```bash
npm test
```

## Building for Production
Create an optimized production build in the `build` directory:
```bash
npm run build
```
